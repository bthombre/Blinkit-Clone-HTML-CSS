# Blinkit Clone UI

This project is a clone of the Blinkit homepage created using HTML and CSS.
The purpose of this project is to practice frontend development, layout design, and responsive UI.

# Features

* Blinkit homepage UI clone
* Responsive layout
* Navigation bar
* Product cards section
* Footer section
* Clean design using Flexbox and Grid

# Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid

# What I Learned

* How to structure HTML layout
* How to use Flexbox and Grid
* How to make responsive UI
* How to organize project files
